Android WIFI_ADB的使用

安装插件

在AndroidStudio中的Plugins中安装插件WIFIADB

使用

选择导航栏中的Tools，在标签中找到WIFIADB Tools，或者使用Ctrl+Alt+2快捷键，选择Connect Manual设置想要调试的ip

如果出现连接失败

失败原因：unable to connect to 192.168.0.115:5555由于目标计算机积极拒绝，无法连接。 (10061)

当连接时 提示这个错误时，按顺序输入命令：

adb usb 
adb kill-server 
adb tcpip 5555 
adb connect 192.168.0.1x:5555

其他使用命令

USB调试模式与WIFI调试模式切换 
adb usb 切换到USB模式 
adb tcpip 5555 切换到WIFI无线调试

WIFI 无线调试下 连接与断开连接的指令 
adb connect 192.168.0.1x 连接设备 
adb disconnect 192.168.0.1x 断开设备连接

1.查看当前连接设备
adb devices
2.安装apk
adb install -r app.apk
adb push app.apk 路径
3.获取文件
adb pull 文件路径 存储位置
4.查看log
adb shell
logcat | grep “TAG”
5.重新挂载分区
adb remount
6.删除apk
adb remount
adb shell
cd apk路径
rm app.apk
7.删除AS直接编译的apk
adb remount
adb shell
cd data/app
rm .apk
8.启动app/activity
adb shell am start -n 包名/类名
9.关闭Activity
adb shell am force-stop 包名
10.发送广播
adb shell am broadcast -a “action”
11.启动service
adb shell am startservice -n 包名/类名 -a action
12.查看全部进程
adb shell ps
ps -A
13.查看指定包名进程
adb shell ps | grep “包名”
14.杀死指定进程
adb shell kill [pid]
15.模拟按键
adb shell input keyevent 3
16.录制屏幕
adb shell screenrecord /sdcard/.mp4
