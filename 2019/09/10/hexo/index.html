hexo 简单使用

1.安装hexo

参考 https://www.jianshu.com/p/201283bcd64a

2.编译部署命令

hexo d -g

3.清除缓存文件 db.json 和已生成的静态文件 public

hexo clean

4.自动生成网站静态文件，并部署到设定的仓库

hexo deploy (hexo d)

5.生成网站静态文件到默认设置的 public 文件夹

hexo generate (hexo g)

6.开启本地服务,用于预览

hexo server (hexo s)
